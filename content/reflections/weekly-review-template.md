# 每周研究复盘

## 1. 本周最重要的新认识

用自己的语言写出真正理解的内容，不复制论文原句。

## 2. 最有信息量的失败

记录被证伪的判断、没有控制好的变量，以及与预期不一致的结果。

## 3. 仍然无法解释的现象

把“感觉哪里不对”改写成明确的问题。

## 4. 下周唯一优先问题

- [ ] 一个需要补充的数据
- [ ] 一个关键对照实验
- [ ] 一个需要重新检查的假设

> 好的复盘不是总结做了多少，而是决定下一步为什么值得做。
