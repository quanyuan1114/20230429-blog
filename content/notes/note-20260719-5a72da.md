## AGENTS.md : 全局持久系统提示文件
- ### 定位：项目级常驻行为约束、项目上下文规则，避免会话间上下文丢失；
- ### 生效时机：每次 Codex 会话初始化自动加载；
- ### 作用：
1. **代码规范**：命名、格式化、Lint、注释、模块拆分规则；
2. **工程命令**：安装依赖、构建、单元 / 集成测试、格式化脚本；
3. **文件访问黑名单**：禁止修改配置、沙箱、遗留底层代码、密钥文件；
4. **变更后置校验**：改完代码必须自动执行测试、格式化、类型检查；
5. **提交 / PR 规范**：commit 格式、新增依赖流程、文档同步要求
### 快捷键:
- shift+enter     自主换行
- /side           打开侧边任务
### 区别README.md 
- `README.md`：写给人类开发者；
- `AGENTS.md`：写给 Codex 等 AI 编码代理，纳入模型输入 Prompt 链。
