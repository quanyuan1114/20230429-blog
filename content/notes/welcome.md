# 学习笔记如何组织

这里不是资料收藏夹，而是一套持续生长的知识系统。

## 每篇笔记尽量回答

- 这个概念解决什么问题？
- 它依赖哪些假设？
- 与相近方法相比有什么差异？
- 哪些结论已经有实验或论文证据？
- 下一步还需要验证什么？

## Markdown 能力

编辑器支持 **GFM 表格**、任务列表、代码块、链接和常见排版。例如：

| 记录对象 | 推荐内容 |
| --- | --- |
| 论文 | 问题、方法、证据、局限 |
| 实验 | 假设、变量、结果、异常 |
| 概念 | 定义、直觉、公式、例子 |

```python
research_note = evidence + reasoning + next_question
```

> 保存会直接生成 GitHub 提交；只有标记为 published 的内容会显示在网站上。
